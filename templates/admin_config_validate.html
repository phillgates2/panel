Configuration Tips
Security
  • • Use strong passwords for database and RCON
  • • Set unique SECRET_KEY in production
  • • Enable SSL/TLS in production environments
  • • Restrict database access to localhost
Database
  • • Use MySQL/PostgreSQL for production
  • • SQLite is suitable for development only
  • • Regular database backups recommended
  • • Monitor connection pool settings
Performance
  • • Redis improves performance significantly
  • • Use production WSGI server (gunicorn)
  • • Enable caching for static assets
  • • Monitor memory usage regularly
Maintenance
  • • Keep dependencies updated
  • • Monitor log files for errors
  • • Test configuration changes in staging
